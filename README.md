Menentukan Bilangan Ganjil atau Genap
​Fungsi Program: Untuk mengidentifikasi apakah bilangan bulat yang dimasukkan pengguna adalah ganjil atau genap.
​Cara Kerja: Program menerima input bilangan bulat (N). Program menggunakan operator modulo (%). Jika N \text{ mod } 2 hasilnya adalah 0, maka bilangan tersebut Genap. Jika hasilnya 1, maka bilangan tersebut Ganjil.
